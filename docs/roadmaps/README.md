# Roadmaps Overview

Last updated: 2026-03-19

## Purpose

This folder contains the detailed roadmaps for the project.

The goal is to keep each roadmap shorter and easier to follow than one large
combined document.

## Which Roadmap To Read

1. Start with `backend_roadmap.md` if you are changing:
   - database tables,
   - booking rules,
   - Supabase,
   - Stripe,
   - payment state,
   - admin-editable event settings.
2. Read `frontend_roadmap.md` if you are changing:
   - public page layout,
   - booking UX,
   - admin page design,
   - responsive behavior.
3. Read `misc_roadmap.md` if you are changing:
   - project structure,
   - documentation layout,
   - ngrok testing,
   - monitoring,
   - deployment,
   - launch preparation.
